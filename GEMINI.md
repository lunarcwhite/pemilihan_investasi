## Project

**SmartInvest**

Aplikasi Rekomendasi Pemilihan Instrumen Investasi Bagi Individu dengan menggunakan metode TOPSIS (Technique for Order of Preference by Similarity to Ideal Solution) untuk mengalkulasi dan meranking opsi investasi berdasarkan kriteria finansial pengguna. Aplikasi ini dirancang mobile-friendly dan berjalan gratis penuh menggunakan Vue 3/Nuxt 3, Supabase, dan Vercel/Netlify.

**Core Value:** Memberikan rekomendasi instrumen investasi yang paling sesuai dengan kondisi finansial spesifik individu menggunakan algoritma TOPSIS secara akurat, cepat, dan mobile-friendly tanpa biaya operasional server (100% free hosting).

### Constraints

- **Tech Stack**: Vue 3 (Vite) / Nuxt 3, Supabase (PostgreSQL, Auth), Tailwind CSS.
- **Hosting Budget**: Rp0 (Vercel/Netlify + Supabase Free Tier).
- **UI Interaction**: Slider C2, C3, C4 dari Kiri (Kecil/Aman/Konservatif/Likuiditas Rendah) ke Kanan (Tinggi/Agresif/Likuiditas Tinggi).
